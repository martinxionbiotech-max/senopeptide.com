# Waste Disposal Procedures

## 1. Waste Classification

| Waste Type | Examples | Container | Label Color |
|---|---|---|---|
| Halogenated solvent waste | DCM, DMF with TFA | Glass or HDPE | Red |
| Non-halogenated solvent waste | ACN, methanol, ether | HDPE | Yellow |
| Aqueous acidic waste | Dilute TFA, piperidine solutions | HDPE | White |
| Solid chemical waste | Used resin, spent drying agent | Sealed bag/container | Black |
| Sharps | Needles, broken glass | Sharps container | Yellow |

## 2. Storage Requirements

| Requirement | Specification |
|---|---|
| Maximum storage | 3 months |
| Containers | Sealed, labeled, compatible with contents |
| Secondary containment | Spill tray for all liquid waste |
| Incompatible segregation | Acids separate from bases, oxidizers separate from flammables |

## 3. Disposal Procedure

1. Segregate waste at point of generation
2. Label container with waste type, contents, date, generator name
3. Store in designated waste area
4. Licensed waste contractor collects on scheduled basis
5. Maintain waste manifest records
