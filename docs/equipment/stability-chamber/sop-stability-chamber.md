# Stability Chamber — Standard Operating Procedure

## 1. Pre-Operation

- [ ] Verify temperature and humidity set points
- [ ] Check chamber door seal integrity
- [ ] Confirm water reservoir level (for humidity control)
- [ ] Verify chart recorder or data logger function

## 2. Loading Procedure

1. Pre-equilibrate chamber to target conditions
2. Load samples in labeled positions
3. Do not block airflow vents
4. Record loading map in study file
5. Set alarm parameters

## 3. Monitoring

| Frequency | Task |
|---|---|
| Continuous | Data logger recording (15 min intervals) |
| Daily | Visual check, temperature/humidity log |
| Weekly | Chart review, alarm test |
| Monthly | Calibration check against NIST reference |

## 4. Sample Pull Schedule

| Condition | Pull Points |
|---|---|
| Accelerated (40 °C/75% RH) | 0, 1, 3, 6 months |
| Intermediate (30 °C/65% RH) | 0, 3, 6 months |
| Long-term (25 °C/60% RH) | 0, 3, 6, 9, 12, 18, 24, 36 months |

## 5. Out-of-Specification

1. If chamber deviates from set point >2 h: initiate deviation report
2. Quarantine affected samples
3. Investigate and document
4. If temporary excursion: include in final stability report
