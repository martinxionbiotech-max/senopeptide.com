# Analytical HPLC System — Technical Specification

!!! note "Official Equipment Specification"
    This document provides the technical specification for analytical HPLC systems used at SENO Biotechnology. For commercial inquiries, visit the [SENO Biotechnology Official Website](https://www.senopeptide.com).

## 1. Technical Parameters

| Parameter | Specification |
|---|---|
| Flow Rate Range | 0.1 – 5.0 mL/min |
| Maximum Operating Pressure | 6000 psi (414 bar) |
| Solvent Delivery | Quaternary or binary gradient |
| Degasser | Online vacuum degassing (4 channels) |
| Injector | Autosampler with sample cooling (4 – 40 °C) |
| Injection Volume | 0.1 – 100 µL |
| Column Compartment | Peltier-controlled, 10 – 85 °C |

## 2. Detector Specifications

| Parameter | DAD (Diode Array) | Specification |
|---|---|---|
| Wavelength Range | 190 – 800 nm | Full spectrum |
| Lamp | Deuterium + Tungsten | Automatic switch |
| Bandwidth | 1 – 40 nm | Adjustable |
| Data Rate | Up to 100 Hz | Real-time spectral acquisition |
| Noise | <±0.5 × 10⁻⁵ AU | At 254 nm |

## 3. System Performance

| Parameter | Specification | Test Method |
|---|---|---|
| Flow Precision | <0.3% RSD | Retention time repeatability |
| Injection Precision | <0.5% RSD | Area repeatability (n=6) |
| Gradient Accuracy | ±0.5% | Composition verification |
| Wavelength Accuracy | ±1 nm | Holmium oxide filter |
| Sensitivity (S/N) | >1000 | 1 µL of 1 µg/mL caffeine |

## 4. Column Specifications

| Parameter | Typical Value |
|---|---|
| Column Dimensions | 150 × 4.6 mm, 250 × 4.6 mm |
| Particle Size | 3 – 5 µm |
| Pore Size | 100 – 300 Å |
| Stationary Phase | C18, C8, C4, HILIC |
