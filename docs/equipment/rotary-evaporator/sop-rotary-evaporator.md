# Rotary Evaporator — Standard Operating Procedure

## 1. Safety Precautions

- Use acid-resistant trap for TFA evaporation
- Ensure vacuum pump has adequate oil or dry pump is verified
- Secure all connections with clamps

## 2. Pre-Operation

- [ ] Check receiving flask is empty and clean
- [ ] Verify bath fluid level and temperature set point
- [ ] Check condenser coolant flow
- [ ] Ensure bump trap is in place

## 3. Operation

1. Fill flask to ≤50% of nominal volume
2. Attach flask to bump trap with clip
3. Lower flask into bath
4. Start rotation at 100 – 150 rpm
5. Apply vacuum gradually
6. Monitor for bumping
7. When evaporation complete, release vacuum
8. Remove flask

## 4. Cleaning

| Step | Solvent | Method |
|---|---|---|
| Rinse | Methanol | Swirl and drain |
| Rinse | DI water | Swirl and drain |
| Drying | Acetone | Rinse, air dry |
