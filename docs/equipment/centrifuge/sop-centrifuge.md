# Centrifuge — Standard Operating Procedure

## 1. Safety Precautions

- Always balance tubes to within 0.1 g
- Never exceed rotor maximum speed
- Use sealed tubes for TFA-containing solutions

## 2. Pre-Operation Checks

- [ ] Rotor is properly installed and locked
- [ ] Temperature set point confirmed
- [ ] Tubes balanced by mass (not volume)
- [ ] Lid seal is clean and intact

## 3. Operation

1. Place balanced tubes symmetrically in rotor
2. Close lid and lock
3. Set speed (× g or rpm), time, temperature
4. Press Start
5. Wait for rotor to reach full stop before opening
6. Remove tubes carefully

## 4. Post-Operation

1. Wipe rotor dry with lint-free cloth
2. Leave lid open to dry
3. Log usage in equipment logbook
