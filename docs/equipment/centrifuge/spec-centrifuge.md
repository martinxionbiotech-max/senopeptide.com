# Centrifuge — Technical Specification

!!! note "Official Equipment Specification"
    For commercial inquiries, visit [SENO Biotechnology Official Website](https://www.senopeptide.com).

## 1. Technical Parameters

| Parameter | Specification |
|---|---|
| Maximum RCF | 5,000 – 15,000 × g |
| Maximum Speed | 6,000 – 10,000 rpm |
| Rotor Type | Swing-bucket or fixed-angle |
| Capacity | 4 × 250 mL or 4 × 500 mL |
| Temperature Range | −10 °C to +30 °C |
| Temperature Accuracy | ±2 °C |
| Timer Range | 1 – 99 min or continuous |
| Noise Level | <60 dB(A) |

## 2. Typical Operating Conditions for Peptide Precipitation

| Parameter | Setting |
|---|---|
| Speed | 3,000 – 4,000 × g |
| Temperature | 4 °C |
| Time | 10 – 15 min |
| Rotor | Swing-bucket, pre-cooled |
| Brake | Medium (to avoid pellet resuspension) |
