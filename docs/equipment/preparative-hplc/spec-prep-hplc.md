# Preparative HPLC System — Technical Specification

!!! note "Official Equipment Specification"
    This document provides the technical specification for preparative HPLC systems used at SENO Biotechnology. For commercial inquiries, visit the [SENO Biotechnology Official Website](https://www.senopeptide.com).

## 1. System Overview

The preparative HPLC systems are designed for high-throughput reversed-phase purification of synthetic peptides. Binary high-pressure gradient systems with dual-wavelength UV detection enable automated fraction collection at user-defined purity thresholds.

## 2. Pump System Specifications

| Parameter | Value | Test Method |
|---|---|---|
| Flow Rate Range | 10 – 200 mL/min | Gravimetric measurement |
| Maximum Operating Pressure | 3000 psi (207 bar) | — |
| Pump Type | Dual-piston series, HPLC-grade heads | — |
| Flow Accuracy | ±1.0% of setpoint | Gravimetric, 10 min collection |
| Flow Precision | ±0.5% RSD | n=6 measurements at 50 mL/min |
| Gradient Composition Accuracy | ±0.5% | Spectrophotometric |
| Gradient Composition Precision | ±0.2% RSD | — |
| Piston Material | Sapphire | — |
| Check Valve Material | Ruby ball / ceramic seat | — |

## 3. UV-Vis Detector Specifications

| Parameter | Value |
|---|---|
| Detector Type | Variable wavelength UV-Vis |
| Wavelength Range | 190 – 800 nm |
| Wavelength Accuracy | ±1 nm (holmium oxide filter verified) |
| Bandwidth | 8 nm (fixed) |
| Light Source | Deuterium (UV) + Tungsten (Vis) |
| Flow Cell | Preparative, 2 mm path length |
| Flow Cell Volume | 10 µL |
| Time Constant | 0.1 – 5.0 s (selectable) |
| Linear Range | Up to 2.0 AU (at 254 nm) |
| Noise | <±0.5 × 10⁻⁵ AU at 254 nm |

## 4. Column and Scale Parameters

| Parameter | Semi-Prep | Preparative |
|---|---|---|

| Column Length | 150 – 250 mm | 250 mm |
| Particle Size | 5 – 10 µm | 10 – 15 µm |
| Pore Size | 100 – 300 Å | 100 – 300 Å |
| Stationary Phase | C18, C8 | C18 |
| Flow Rate | 5 – 20 mL/min | 50 – 200 mL/min |
| Typical Load | 1 – 50 mg | 100 mg – 5 g |
| Injection Volume | 0.1 – 5 mL | 1 – 50 mL |

## 5. Injection and Fraction Collection

| Component | Specification |
|---|---|
| Injector | Manual injection valve (Rheodyne) or automated injector |
| Injection Loop | 1 mL, 5 mL, 10 mL, 50 mL (interchangeable) |
| Fraction Collector | Automated, time-based or UV threshold-based |
| Collection Threshold | User-selectable (default: ≥50 mAU at 220 nm) |
| Fraction Racks | Test tubes 15 mL, or bottles 250 mL |

| Utility | Specification |
