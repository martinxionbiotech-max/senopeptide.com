# SDP™ Soluble Dimer Protein Platform — Technical Description

## 1. Platform Overview

The SDP™ (Soluble Dimer Protein) platform enables the creation of soluble protein constructs that mimic natural protein-protein interactions. The constructs are designed to present target proteins in a dimeric configuration for improved binding performance in screening assays.

## 2. Technical Specifications

| Parameter | Specification |
|---|---|
| Protein Configuration | Soluble dimeric construct |
| Expression System | Mammalian (HEK293) |
| Tag Options | Fc, His, Biotin (enzymatic) |
| Compatible Assays | ELISA, SPR (Biacore), BLI (Octet) |
| Purity | ≥90% by SEC-HPLC |
| Endotoxin | <1.0 EU/mg |
| Format | Liquid or lyophilized |

## 3. Applications

- Receptor-ligand interaction modeling
- Antibody screening and characterization
- Epitope mapping studies
- Biomolecular interaction analysis
