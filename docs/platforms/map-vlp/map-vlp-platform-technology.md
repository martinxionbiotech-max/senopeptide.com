# MAP-VLP Membrane Associated Protein VLP Platform — Technical Description

## 1. Platform Overview

The MAP-VLP (Membrane Associated Protein Virus-Like Particle) platform enables expression of full-length multi-pass membrane proteins incorporated into virus-like particle systems. The platform maintains native protein structure and lipid-bilayer presentation.

## 2. Technical Specifications

| Parameter | Specification |
|---|---|
| Membrane Protein Type | Multi-pass transmembrane proteins |
| Expression System | Mammalian (HEK293) |
| Display | VLP surface incorporation |
| Membrane Environment | Native-like lipid bilayer |
| Purity | ≥85% by SDS-PAGE |
| Particle Size | 100 – 200 nm typical |
| Density | High-density target display |
| Storage | −80 °C or liquid N₂ |

## 3. Advantages

- Full-length protein expression (not truncated or solubilized)
- Native-like membrane environment for structural integrity
- Compatible with structural biology, immunogenicity studies
- High-density target display on VLP surface
